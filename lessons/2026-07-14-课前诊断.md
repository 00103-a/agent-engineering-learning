## B. 执行推演（建议 25 分钟）

阅读下面代码，不运行，先写出预测：

```python
import asyncio


async def work(name: str, delay: float, fail: bool = False) -> str:
    await asyncio.sleep(delay)
    if fail:
        raise RuntimeError(name)
    return name


async def main() -> None:
    tasks = [
        work("weather", 0.2),
        work("course", 0.1, fail=True),
        work("news", 0.3),
    ]
    result = await asyncio.gather(*tasks)
    print(result)
```

回答：

1. 是否会执行 `print(result)`？
2. `course` 失败后，另外两个协程一定会被取消吗？
3. 如果希望收集每个调用的成功或失败结果，可以怎样修改？
4. 如果这些调用包含副作用，直接并发执行还需要考虑什么？
